## 附录：Harness Engineering 文章精读

以下 2 篇是 Harness Engineering 方法论的核心文献：

### 1. Effective harnesses for long-running agents
**发布时间：** 2025-11-26  
**链接：** https://www.anthropic.com/engineering/effective-harnesses-for-long-running-agents  
**摘要：** 探讨长时间运行 AI Agent 的 harness 设计模式——如何管理 agent 的执行周期、如何处理中断与恢复、如何设计检查点机制。

### 2. Harness design for long-running application development
**发布时间：** 2026-03-24  
**链接：** https://www.anthropic.com/engineering/harness-design-long-running-apps  
**摘要：** 深入讲解应用开发场景下的 harness 设计——将 Agent 的推理能力与具体业务流程解耦，构建可预测、可控的 AI 应用骨架。

> **什么是 Harness？**  
> 在 Anthropic 的 Agent 工程体系中，"Harness"（ harness 原意"马具/安全带）指的是约束和管理 AI Agent 行为的运行时框架。它定义了：Agent 何时该调用工具、如何管理多步骤任务的生命周期、如何处理错误和重试、以及如何在受限环境中安全运行。
