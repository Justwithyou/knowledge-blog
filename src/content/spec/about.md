# 关于本站

欢迎来到 **技术漫客**！

这是一个基于 [Astro](https://astro.build/) 框架和 [Fuwari](https://github.com/saicaca/fuwari) 主题构建的个人技术博客，专注于：

- **前端开发**：Vue.js、React、Astro 等框架实践
- **后端技术**：Node.js、Python、Go 等服务端开发
- **DevOps**：Docker、Nginx、CI/CD 等运维实践
- **AI 与机器学习**：大模型应用、自动化工具探索
- **编程心得**：日常开发中的踩坑记录与经验总结

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Astro 5.x |
| 主题 | Fuwari |
| 样式 | Tailwind CSS |
| 部署 | 静态资源 + Nginx |
